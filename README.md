# 🚀 How to Run

### Clone the repository

```bash
git clone https://github.com/jessicabhupathi/FUTURE_FS_01.git
```

### Open the project folder

```bash
cd FUTURE_FS_01
```

### Run the website

Simply open **index.html**

OR

Open the folder in **Visual Studio Code** and launch it using **Live Server**.
